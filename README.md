# Inventario PYME — Demo

Aplicación web demostrativa para gestión de inventario, ventas, compras y reportes. Está pensada como demo para clientes: no requiere backend y los datos se almacenan localmente en el navegador.

**Cómo probar (cliente)**

- Instale dependencias e inicie la aplicación localmente:

```bash
npm install
npm run dev
```

- Abra la URL que muestre Vite (por defecto http://localhost:5173).

**Credenciales de demo**

- Usuario: `demo`
- Contraseña: `pyme2026`

La sesión se guarda en `sessionStorage` (al cerrar la pestaña se pierde). Los datos de la demo (inventario, ventas, compras) se guardan en `localStorage` del navegador.

**Notas de seguridad y uso**

- Esta es una demo orientativa: no utilice credenciales reales ni datos sensibles.
- No hay integración con servicios externos ni exposición de claves en este repositorio.
